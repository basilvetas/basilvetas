# basilvetas.com (WIP)

## Project setup
```
yarn install
```

### Compile and hot-reload for development
```
yarn serve
```

### Compile and minify to /dist for production
```
yarn build
```

### Lint and fix files
```
yarn lint
```

### Initialize Firebase
```
firebase init
```

### Test with Firebase locally pre-production
```
firebase serve
```

### Deploy /dist to Firebase for production
```
firebase deploy
```
